dotfiles
==================================================
Personal collection of Unix configuration files and a script for creating symbolic links with backup of existing dotfiles.

Installation
------------
```bash
$ cd ~
$ git clone https://github.com/tavizon/dotfiles.git
$ cd dotfiles
$ ./linkdotfiles
```
